# Real Estate Sales Analysis Dashboard (Tableau)

## Project Overview
This project analyzes real estate sales trends from 2011 to 2022 using Tableau.

## Dashboard Features
- Total Sales by Town
- Monthly Sales Trend
- Yearly Sales Trend
- Property Type Distribution
- Geographic Sales Distribution
